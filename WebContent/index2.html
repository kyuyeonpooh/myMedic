myMedic


myMedic is a dApp which can issue hospital related documents such as Medical Certificates, Prescriptions and Medical Confirmations. By using "BlockChain", myMedic prevents documents from being falsified.

Security


We enhanced security by adopting "BlockChain" technology. Issued documents from myMedic are not able to be falsified. This explains the availability of myMedic.

Cost


Does not require data storage like DataBase, because myMedic stores documents by XML on the Web. It not only reduces the fee which paid by patients, but leads to cost reduction for institutions.
